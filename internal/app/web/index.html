Add IPv6 Assignment

Record which IPv6 address or prefix a target side may use from a parent prefix. This does not mean adding the address onto the host-side target interface.

/128 means a single IPv6 for the target to use and can be announced with managed RA + DHCPv6 IA_NA; /64 suits a guest subnet and will advertise that prefix with RA for SLAAC; other prefix lengths are better treated as delegated downstream prefixes. The address is not bound to the host-side target interface.
